# 用 AI 在工作中提效（OnlyRouter 场景速查）

> 这份是给群助手机器人的知识底座：当同事问"这个活能不能用 AI 干""怎么让 AI 帮我做 X"，据此给出能马上照做的方法。

## 回答提效问题的第一原则：先让非技术同事用 Codex 上手

大多数问提效的是**非技术同事**，回答时**先给他们能立刻上手的做法——用 Codex 桌面工具**（装好后用大白话对话就能干活，不用写代码）。**不要一上来就讲写脚本、调 API**，那会劝退人。

标准回答结构：
1. **先（人人可用）**：告诉他们用 Codex 怎么做——给一段可以直接复制发给 Codex 的话。
2. **后（进阶/规模化）**：如果这活量大、要批量或自动化，再补上 API/脚本方案（面向开发或有技术能力的人，这部分可以讲深）。

例：有人问"能不能用 AI 帮我筛简历"——
> 先说：可以，装个 Codex（找管理员要 OnlyRouter Switch 一键配好），然后把简历丢给它，发这样一句话：「这是 20 份简历（附文件），帮我按『3年以上后端经验+会 Go』筛一遍，列出符合的人和理由，做成表格」。
> 再说：如果每天几百份要自动化处理，可以走 API 批量跑，这块找开发或我再细讲。

## 选模型的一句话原则

- **日常对话、写东西、改方案**：`gpt-5.5-ab`（便宜好用）或 `deepseek-v4-pro`（更省）。
- **要最强推理/复杂长文**：`claude-opus-4-8-ab` 或 `gpt-5.5`。
- **写代码**：`claude-sonnet-4-6-ab`（性价比）、`claude-opus-4-8-ab`（最强）、`kimi-k2.7-code`（编码专优）。
- **画图**：`qwen-image-2.0`（便宜快，约 ¥0.2/张）、`doubao-seedream-5-0`、`gpt-image-2`（质量高）。
- **文字转语音 TTS**：`qwen3-tts-flash`（便宜）、`doubao-tts-2.0`。
- **录音/视频转文字**：`whisper-1`、`qwen3-asr-flash`、`gpt-4o-transcribe`。
- **视频生成**：`sora-2`、`doubao-seedance-2-0`。

完整实时列表：机器人会自带最新的，也可去 onlyrouter.ai/models 查。

## 高频提效场景 + 可直接抄的提示词

### 写周报 / 工作总结
把零散记录丢给它，让它归纳：
> 下面是我这周做的事的流水记录，帮我整理成一份条理清晰的周报，分「本周完成 / 进行中 / 下周计划」三块，每条一句话说清价值，不要空话：[粘贴你的流水]

### 写/改邮件、消息
> 帮我写一封给客户的邮件：目的是 X，对方背景是 Y，语气正式但不生硬，200 字以内。

### 整理 / 分析表格数据
> 这是一份 CSV 数据（粘贴），帮我：1) 找出销售额前三的地区；2) 算每个月的环比增长；3) 用一段话总结趋势。

### 翻译
> 把下面这段翻成英文，风格偏商务正式，保留专业术语：[原文]

### 会议录音转文字 + 提炼纪要
先用 `whisper-1` 转写，再让对话模型提炼：
> 这是会议转写稿（粘贴），帮我提炼成纪要：决议事项、待办（谁负责、截止时间）、遗留问题。

### 调研 / 梳理知识
> 我要了解 X 领域，帮我用结构化方式讲清楚：核心概念、主流方案对比、常见坑。面向没背景的人，说人话。

### 生成图片配图
让 AI 帮忙调接口（非开发同事用 Codex，一句话让它代劳）：
> 帮我生成一张图片：调用 onlyrouter.ai 的图片接口，model 用 qwen-image-2.0，描述是「一只橘猫戴安全帽在工地搬砖，卡通风格」，下载到桌面。

## 用 AI 的通用心法（可转告用户）

- **给背景、给例子、给格式要求**：说清你是谁、给谁看、要什么风格和长度，比只说"帮我写个东西"效果好十倍。
- **不满意就追问**：AI 第一版不对很正常，直接说"太正式了，再口语一点""第二点展开讲"，多轮比一次到位更省事。
- **让它先出框架再填充**：长东西先让它列大纲，你确认方向再让它写，避免跑偏白写。
- **别把敏感信息（真实 Key、客户隐私、内部机密）贴进去**：尤其别把 API Key 发群里。

## 面向"非开发同事"的入口

不会写代码的同事，推荐装 **Codex** 桌面工具接 OnlyRouter，用大白话对话就能写文案、画图、转语音、录音转文字。详细配置见《非开发人员使用教程-Codex接入》。
